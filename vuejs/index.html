# Crear un projecto VueJS

# Instalación con npm

npm install -g @vue/cli

# VueJS

Para iniciar un nuevo proyecto con VuePress se abre una terminal en el directorio donde queremos guardar nuestro desarrollo y se ejecuta:

vue create <nombredelproyecto>

El nombre del proyecto no acepta mayúsculas. VueJS también puede manejar los proyectos desde un navegador. Para esto se debe iniciar un proyecto con:

vue ui

Se abre el navegador y se crea el proyecto y se gestiona desde ahí.
