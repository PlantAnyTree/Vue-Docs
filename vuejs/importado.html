# Vuejs: CLI

# Introducción

Vuejs CLI proporciona un sistema completo para el desarrollo de aplicaciones. Se instala junto a [[wiki:programacion:nodejs:instalacion|Nodejs]] y [[wiki:programacion:nodejs:npm|npm]].
Existen muchas aplicaciones o paquetes que proporcionan ayudas al desarrollo con Vuejs.

# Instalación

Después de instalar [[wiki:programacion:nodejs:instalacion|Nodejs]] y [[wiki:programacion:nodejs:npm|npm]] ejecutamos el comando:
sudo npm install -g @vue/cli Esto instala Vue/CLI de forma global en nuestro sistema.
Para comprobar la versión que hemos instalado ejecutamos en la consola:
vue --version Si en un futuro queremos actualizar el paquete debemos ejecutar el comando:
npm update -g @vue/cli

# Crear un proyecto

Para crear un proyecto nuevo, se ejecuta desde la terminal el comando:
vue create componentes El nombre del proyecto debe estar en minúsculas. Aparece un menú par seleccionar instalaciones prefabricadas o para crear una nueva instalación seleccionando manualmente los paquetes a instalar. Por defecto la instalación viene con babel y eslint.
Una vez creado el proyecto muestra unas indicaciones para ejecutar el proyecto:

 $ cd componentes
 $ npm run serve

El proyecto contiene los directorios public, src y node_modules, además de los archivos babel.config.js, package.json, package-lock.json y README.md.

# Nuevo proyecto con GUI

[[wiki:programacion:vuejs:cli:proyecto]] Desde nuestro GUI podemos gestionar los proyectos de Vuejs. Pero primero se debe ejectuar desde la terminal el comando:
vue ui Fuente oficial de Vuejs

# Añadir paquetes a nuestro proyecto

Una vez creado el proyecto, podemos necesitar instalar algún paquete adicional con el comando:
vue add eslint Este comando puede modificar archivos ya existentes en nuestro proyecto, por lo que es recomendable hacer una copia de seguridad primero.
