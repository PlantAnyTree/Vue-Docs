# Configuración de VuePress

La configuración se establece en el archivo /docs/.vuepress/config.js. Consta principalmente de dos partes, una general del proyecto y otra de la plantilla (theme).

# Configuración general

En esta parte se determinan los siguientes elementos:

# title

El título del proyecto

title: "Nombre del Proyecto",

# description

Un pequeño resumen del objeto del proyecto:

description: "Aquí se detalla un poco más el proyecto",

# Idioma

El proyecto puede ser multilíngüe, pero en este caso, lo que quiero es que en la página de inicio cargue el idioma español como el idioma en el que está escrito el proyecto:

    locales: {
        '/': {
          lang: 'es-ES', // this will be set as the lang attribute on <html>
          title: 'VuePress',
          description: 'Generador de sitios web basado en Vuejs'
        }
    
}

De esta manera en la página principal aparece:

<html lang="es-ES"><head>

Una confiruraión básica, sería entonce:

module.exports = {
    locales: {
        '/': {
          lang: 'es-ES', // this will be set as the lang attribute on <html>
          title: 'VuePress',
          description: 'Generador de sitios web basado en Vuejs'
        }
 }
}
