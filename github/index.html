# Crear una repo con GitHub

# Instalación del paquete git

El paquete git está disponible en la mayoría de las distribuciones Linux, por lo tanto es muy fácil de instalar. En Debian y sus derivados se ejecuta en la terminal:

sudo apt-get install git

# Crear y configurar el .gitignore

Dentro del directorio del proyecto creo un archivo con el nombre .gitignore:

nano .gitignore

Dentro de ese achivo escribo los archivos y directorios que no quiero que se sincronicen con git:

node_modules
docs/.vuepress/dist

# Nuevo repositorio en GitHub

Una forma de hacerlo es entrando en la web de GitHub. Una vez logeado creo el repositorio y lo dejo en blanco, ni siquiera creo el README.md. Para facilitar la actualización necesito una clave ssh. Puedo comprobar si ya tengo una en mi equipo:

ls -ahl ~/.ssh

Si no tengo ninguna clave del tipo 'id_rsa.pub' la creo con:

$ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

Abro el archivo generado con la extensión .pub. copio el contenido y lo añado a la sección de claves SSH de GitHub
